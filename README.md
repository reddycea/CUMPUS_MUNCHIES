# CUMPUS_MUNCHIES
a full-stack campus food ordering system called Campus Munchies, built using Flask (Python). It’s essentially like a mini version of Uber Eats or Bolt Food, but designed for a university environment.
User authentication (login/register)
Multi-store food browsing
Cart and checkout system
Multiple payment methods (Stripe, PayPal, M-Pesa, cash)
Order tracking and history
Notifications (email/SMS)
Admin & super admin dashboards
